
 **SHOP.CO: Modern E-Commerce Front-End Project**

 **Project Overview**

**SHOP.CO** is a modern, fully **responsive front-end e-commerce website UI**, developed as a final project by web development interns at Buildables. The interface is inspired by a professional **Figma design**, focusing on delivering a **clean, stylish, and intuitive user experience** across all devices.

This project is a strong showcase of proficiency in building complex layouts using modern web standards.

---

 **Key Features & Page Structure**

The project provides a comprehensive e-commerce browsing experience, structured across the following key pages and components:

 **Home Page (`index.html`)**

The main entry point, designed to drive sales and engagement:

* **Hero Section:** A prominent banner highlighting the brand, complete with key statistics (**200+ Brands, 2,000+ Products, 30,000+ Customers**) and popular brand logos (Versace, Zara, Gucci, Prada, Calvin Klein).
* **New Arrivals & Top Selling:** Dedicated sections using a modern **grid layout** to showcase fresh and popular products.
* **Browse by Dress Style:** A grid section allowing users to explore products categorized by style (Casual, Formal, Party, Gym).
* **Customer Testimonials:** A dedicated review section featuring a **smooth scrolling/carousel-style layout**.
* **Newsletter Subscription:** A prominent banner to encourage user sign-ups for updates and offers.

 **Category/Product Listing Page (`category.html`)**

* Allows users to view products filtered by style or type.
* Features a responsive display of various items.

 **Product Detail Page (`product-detail.html`)**

* Provides in-depth information about a single item.
* **Add to Cart Functionality (UI/UX):** Features an interactive component for selecting size/quantity and adding the item to the cart (**front-end state only**).

 **Cart Page (`cart.html`)**

* A dedicated view for managing selected products before checkout.

**Global Navigation**

A cohesive and user-friendly experience maintained throughout with a comprehensive **Footer Navigation** including Company Info, Help Section, FAQs, and Quick Links.

---

 **Technologies Used**

This is a pure front-end project demonstrating strong foundational development skills:

 **HTML5** - Semantic page structure and organization.
 
 **CSS3** - Advanced styling, custom CSS variables, **Grid Layout**, and **Media Queries** for fully **responsive design**.
 
 **JavaScript (ES6)** - Interactivity, dynamic behaviors, and managing simple front-end state (e.g., "Add to Cart" UI updates). 

> **Note:** This is a **front-end only** project. No backend or server setup is required, and all dynamic features are managed through client-side JavaScript.
